--- Default Controls (Case Insensitive) ---
| Action     | Player 1 | Player 2 |
|------------|----------|----------|
| UP         | W        | I        |
| DOWN       | X        | M        |
| LEFT       | A        | J        |
| RIGHT      | D        | L        |
| STAY       | S        | K        |
| DISPOSE    | E        | O        |
* Note: Controls can be changed in the Settings menu.


Notice:

1) The three example recordings are in the folder called "examples"

2) added -load -interactable argument support to adjust recording speed up to 16x. -load only locks it at normal speed.

3) Bresenham's Line Algorithm logic used for Torch/Bomb Line-of-Sight.

4) Console color handling adapted from Windows API documentation.
