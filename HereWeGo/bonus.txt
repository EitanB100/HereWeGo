================================================================================
                                BONUS FEATURES
================================================================================

1. Shielded Bomb Explosions 
	Implemented Wall shielding from explosions

2. New Game Element: Potions
	Adds 5 HP until maximum reached.
	marked with P in game.

3. Color Support with Toggle

4. Key Rebinding System
   - Inside Settings, a full key rebinding system that allows players to rebind controls for 
     both Player 1 and Player 2 at runtime (includes conflict detection and prevents them).

5. added -interactable argument
	The argument allows for a fast-forwarding of the recording in various speeds with a progress bar

6. Added functionality to continuous recording of a saved state
	i.e. if you play on -save, save state, relaunch the program on -save and open the same state, it continues recording from the last step.

7. saved state of changed keys
